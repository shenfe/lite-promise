A lightweight (< 100 LOC in main program) implementation of [Promises/A+](https://promisesaplus.com/) (indexed on 2017/12/23).

## Guide

```bash
npm install --save @hengwu/promises-aplus
```

## Implementation

### Prototype Methods

* then
* catch
* done
* finally
* resolve
* reject

### Static Methods

* resolve, resolved
* reject, rejected
* deferred
* all
* race
